# 🖥️ Portafolio IT & Redes | Estilo Terminal

Portafolio profesional desarrollado para un perfil Senior en Infraestructura IT, Telecomunicaciones y Mantenimiento Electromecánico. El diseño rompe con las plantillas web tradicionales e implementa una interfaz de usuario 'Brutalista' y plana, inspirada en paneles de telemetría de servidores y registros de datos (Data Logs).

## 🛠️ Stack Tecnológico

- **React 18**
- **TypeScript**
- **Vite** (para un empaquetado ultrarrápido)
- **Tailwind CSS** (estilizado utilitario hiper-personalizado)
- **Lucide React** (iconografía)

## 💡 Características de la UI/UX

- **Tema "Emerald & Zinc":** Paleta de colores basada en placas de circuito y chasis de hardware.
- **Flujo de Lectura "Data Log":** Eliminación de tarjetas UI convencionales a favor de tablas de registro plano para maximizar la densidad de información técnica.
- **Responsive Design:** Adaptabilidad total bajo el enfoque Mobile-First.

## 🚀 Instalación y Uso

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Ejecuta el servidor de desarrollo en local:
   ```bash
   npm run dev
   ```
